# PinMe Gallery

一时兴起用 PinMe 部署的各种页面

## 🚀 快速开始

```bash
# 安装依赖
npm install

# 本地开发
npm run dev

# 构建
npm run build

# 预览构建结果
npm run preview
```

## 📦 部署到 Vercel

1. 在 [Vercel Dashboard](https://vercel.com/dashboard) 点击 "Add New Project"
2. 选择 "Import Git Repository"
3. 选择 `ruijayfeng/pinme-gallery` 仓库
4. Vercel 会自动检测 Astro 项目并配置构建设置
5. 点击 "Deploy" 完成部署

## 🎨 设计风格

- Anti-Slop 设计：反对平庸、反对套路
- 柔和对比：不刺眼，舒适阅读
- 大量留白：呼吸感强，不拥挤
- 高级字体：Playfair Display + Inter
- 弹性动画：GSAP + ScrollTrigger
